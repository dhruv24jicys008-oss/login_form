# Styled Login Form

1. A simple and responsive HTML & CSS login form with basic JavaScript handling.
2. The form demonstrates modern styling, input fields, and client-side redirection on submit.

# Features

1. Clean and modern login UI

2. Responsive center-aligned layout

3. Username & password input fields

4. Gender selection using radio buttons

5. “Remember Me” checkbox

6. Comments textarea

7. Styled submit and reset buttons

8. JavaScript-based redirection on form submission

# Technologies Used

1. HTML5 – Structure of the form

2. CSS3 – Styling and layout

3. JavaScript – Form submission handling & redirection
